
## Description

Technical test app for SmartSoft.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# local mode
$ npm run start:local
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Support

```bash
# documentation
$ npm run compodoc
```
